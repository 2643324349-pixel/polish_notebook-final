# 波兰语变格单词本 - 产品概述

## 1. 产品概述

- **一句话描述**：面向波兰语学习者的Web/APP工具，通过表格管理词性变格，内置规则引擎自动生成变格/例句，支持双击隐藏单元格自测记忆，并提供变格含义解释页。

- **目标用户**：国内和海外波兰语学习者。

- **核心平台**：Web端 + 手机APP（响应式设计，兼容桌面/平板/手机浏览器）。

---

## 2. 账号系统

### 2.1 注册/登录
- 方式：邮箱 + 密码 / Magic Link（基于 Supabase Auth）
- 区域策略：海外实例优先，MVP跑通后一键复制结构创建国内实例
- **数据隔离**：Notebook表含 `region` 字段（`global` / `cn`），RLS策略自动注入

### 2.2 未登录状态
- 可访问首页介绍和变格含义解释页
- 点击"Sign In"跳转登录页（界面简洁大方，国际化）
- 未登录用户可以创建笔记本，创建页，创建列和行， 但是创建超过5行单词后，创建第六行单词弹出登陆页面，登录之后可以随便创建行， 当用户离开页面，自动保存当前列表内容

### 2.3 登录后
- 自动进入个人笔记本列表页

---

## 3. 变格含义解释页

### 3.1 访问方式
- 侧边栏/底部导航增加「📖 变格指南」入口
- 独立页面，不依赖登录状态

### 3.2 内容结构
- 按词性分类（名词/形容词/代词/数词/动词）
- 每个格展示：名称、用法说明、例句、变格示例表格
- 多语言支持（跟随UI语言）

---

## 4. 非功能性要求

### 4.1 国际化
- 默认语言：英文
- 支持：中文、英文、德文、乌克兰文
- i18n资源文件独立管理（JSON/YAML）
- 法律条款：海外全英文，国内全中文

### 4.2 性能
- 表格渲染100行以内无卡顿
- 规则引擎：Top 2000词秒开，2000+词首次查询100-300ms延迟

### 4.3 合规
- 页脚提供隐私政策与服务条款链接
- 多语言文案预留位置
- 苹果上架需提供"技术支持网址"和隐私政策链接

### 4.4 架构
- **React** 跨平台框架，一套代码编译到 Web / iOS / Android
- **Monorepo** 架构，核心 UI 与业务逻辑复用
- **Supabase** 作为后端服务（Auth + PostgreSQL + Storage）
- 通过环境变量区分 `REGION=GLOBAL` / `REGION=CN`
- 动态注入不同 Supabase 实例 URL 和 Auth 配置
- 双实例 Schema 严格一致，Migration脚本同步执行

### 4.5 技术栈明细（React 版）
- 前端框架：React 18 + TypeScript
- 样式方案：Tailwind CSS（基于 Figma 导出的设计 tokens）

- UI 组件库：shadcn/ui（与 Figma 设计系统匹配）
- 状态管理：Zustand（轻量） + React Query（服务端状态）
- 路由：React Router v6
- 后端：Supabase (PostgreSQL + Auth + Storage)
- 规则引擎：Morfeusz（通过 REST API 调用，或部署为 Supabase Edge Function）
- 表单校验：React Hook Form + Zod
- 拖拽排序：dnd-kit（Web 拖拽） + 移动端手势适配
- 国际化：i18next + react-i18next
- 包管理：pnpm (Monorepo)
- 构建工具：Vite
- 上架方案（后期）：PWA（快速） 或 Capacitor（打包原生）

## 5. Web与移动端操作对照

| 操作方式 | Web 桌面端 | 移动端 | 说明 |
